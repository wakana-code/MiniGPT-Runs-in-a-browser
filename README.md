# 🧠 NanoGPT in Browser

ブラウザで完全に動く nanoGPT。Transformer をフルスクラッチで TypeScript 実装。
npx plugins add vercel/vercel-plugin

上記URLからアクセス可能！

## ✨ 特徴
- ブラウザだけで学習・推論ができる
- Multi-head attention + LayerNorm + GELU を手書き実装
- Adam optimizer + 勾配クリッピング
- Sibugaki AI 会話データセット付属

## 🚀 使い方

「初期化」した後に「学習開始」を押す。学習途中にメッセージを送ることもできるので成長の変遷を楽しみましょう！
